# sas

A new Flutter project.

## Getting Started

This project is a starting point for a Flutter application.

This project is a Flutter quiz application built with a data-driven architecture. The quiz content is loaded from JSON, allowing easy customization without changing the UI code. The app includes audio support, smooth animations, score tracking, and responsive layouts to provide an engaging learning experience. State management is implemented using Provider/Riverpod/BLoC for efficient and scalable application development.
